# 줄거리 맞춤 역사화 슬라이드 영상

해당 시간대에 띄우는 슬라이드형 영상(mp4)을 자동으로 만듭니다.

## 사용법

### 0) 준비
```bash
pip install Pillow imageio-ffmpeg requests
```

### 1) 역사화 다운로드 — 위키미디어 공용
```bash
python slideshow/build.py --fetch
```
- `slideshow/img/01.jpg ~ 22.jpg` 로 저장됩니다.
- **네트워크 허용 목록에 아래 두 호스트가 있어야 동작합니다:**
  - `commons.wikimedia.org`
  - `upload.wikimedia.org`
- 마음에 안 드는 그림은 같은 번호(`07.jpg` 등)로 직접 교체하면 됩니다.
  (직접 받은 그림을 그 번호로 덮어쓰기만 하면 됨)

### 2) 영상 합치기
```bash
python slideshow/build.py --render \
  --audio "내려받은_음성.mp3" \
  --out mary-slideshow.mp4
```
- 각 장면은 페이드 인/아웃으로 부드럽게 전환되고, 하단에 작품명 자막이 붙습니다.
- 오디오 길이에 맞춰 전체 타임라인이 자동 정렬됩니다.

## 그림을 바꾸거나 추가하려면
`build.py` 의 `CHAPTERS` 목록에서 검색어·자막을 수정하면 됩니다.
`("검색어", "화면 자막")` 형식이며, 정확한 파일을 지정하려면
`("file=공용파일명.jpg", "자막")` 처럼 적습니다.
